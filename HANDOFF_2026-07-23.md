## 2. 用户最新要求

用户明确要求 Experience 的室内→室外转场必须是“镜头推进 / 穿越场景”，不是普通纵向换页：

- Experience 整个场景固定在视口中，由同一段滚动进度控制。
- 图 1 持续放大，缩放中心位于拱门内的室外天空，而不是图片中心。
- 人物、柱子、文字、月亮和楼梯随镜头放大自然移出视口。
- 室外图预先铺在同一个固定舞台后方，通过 opacity/交叉淡化逐渐显现。
- 室外图与拱门内天空、地平线和树木大致对齐，形成同一空间。
- 绝对不能让室外图从屏幕底部上移，不能有两张图片上下拼接的横向接缝。
- 室外完全占满视口后，数据随滚动依次浮现；动画结束后才解除 pin 并进入后续内容。

验收重点：

1. 看不到室外图从底部进入。
2. 看不到横向接缝。
3. 滚动感觉是镜头向画面深处推进。
4. 图 1 能平滑推进到用户参考的图 2 近景构图。
5. 最后自然进入同一空间中的室外景。

## 3. 已完成的最新实现

### 固定舞台

Experience 仍使用一条 GSAP 时间轴：

- 触发器：`.experience-scroll`
- 固定元素：`.experience-stage`
- `start: "top top"`
- `end: "bottom bottom"`
- `scrub: 0.1`
- `pin: ".experience-stage"`
- 桌面滚动高度：`470vh`
- 手机滚动高度：`410vh`

动画完成后 ScrollTrigger 才解除固定并进入后续艺术插页 II。

### 两个重叠的全屏图层

Experience JSX 当前是同一 `.experience-stage` 内的绝对定位重叠层，不是上下排列的两个 section：

```tsx
<div className="experience-outdoor">...</div>
<div className="experience-art experience-art--indoor">...</div>
<div className="experience-indoor-copy">...</div>
```

- `.experience-outdoor`：z0，始终原位铺在后方。
- `.experience-art--indoor`：z1，包含室内底图、楼梯、雕像、月亮和左边前景柱。
- `.experience-indoor-copy`：z8，包含 Experience 顶部信息和主标题，与室内镜头同步缩放。
- `.metrics` 和 `.experience-finale` 保持在镜头层之外，用于室外阶段的数据展示。

### 镜头推进时间轴

当前 `src/components/Portfolio.tsx` 的关键参数：

```ts
.fromTo(
  ".experience-statue",
  { xPercent: () => window.innerWidth <= 640 ? -100 : -150, yPercent: 5, opacity: 1 },
  { xPercent: 0, yPercent: 0, opacity: 1, duration: 0.34, ease: "none" },
  0.02,
)
.fromTo(
  ".experience-art--indoor",
  { scale: 1, autoAlpha: 1 },
  { scale: 2.35, duration: 0.42, ease: "none" },
  0.32,
)
.fromTo(
  ".experience-indoor-copy",
  { scale: 1, autoAlpha: 1 },
  { scale: 2.35, duration: 0.42, ease: "none" },
  0.32,
)
.to(".experience-indoor-copy", { autoAlpha: 0, duration: 0.18, ease: "none" }, 0.52)
.fromTo(
  ".experience-outdoor",
  { autoAlpha: 0, scale: 1.08 },
  { autoAlpha: 1, scale: 1.28, duration: 0.2, ease: "none" },
  0.54,
)
.to(".experience-art--indoor", { autoAlpha: 0, duration: 0.18, ease: "none" }, 0.56)
```

重要：Experience 场景时间轴中已经没有以下旧代码：

```ts
.to(".experience-art--indoor", { yPercent: -100 })
.fromTo(".experience-outdoor", { yPercent: 100 }, { yPercent: 0 })
```

不要把它们恢复。当前只有雕像和数据自身使用 `yPercent`，它们不是场景级纵向换页。

### 缩放中心

桌面：

```css

  transform-origin: 64% 38%;
```
