# 媒体素材替换与隐私检查

当前网站未使用外部图片或热链；所有视觉图片位均为本地 CSS 占位。正式替换前，
请把素材复制到以下目录，并采用推荐文件名。

`art/` 内的石膏胸像、石膏手臂和蓝色暮光油画是为本网站生成的原创艺术过渡素材。
页面引用的是压缩 WebP 文件；同名 PNG 是高质量源文件，便于以后重新裁切或导出。

## 推荐文件

### Experience 照片

- `photos/livestream-scene.webp`
- `photos/editing-workflow.webp`
- `photos/account-analytics.webp`
- `photos/publishing-screen.webp`

建议导出 WebP，长边 1600px 左右，同时保留原始文件备份。页面中应填写明确的
宽高属性和准确 alt 文本。

### Experience 视频

- `videos/livestream-highlight.mp4`
- `videos/content-workflow.mp4`
- `videos/livestream-poster.webp`
- `videos/content-workflow-poster.webp`

视频需要 poster，默认静音，不自动播放带声音的内容，并在接近视口时再加载。

### 社交账号图片

- `social/douyin-profile.webp`
- `social/xiaohongshu-profile.webp`

## 公开前必须逐项检查

- [ ] 无无关人员的人脸；如有，已取得许可或完成不可逆打码
- [ ] 无不准备公开的用户名、用户 ID 或头像
- [ ] 无聊天记录、私信、评论区中的个人信息
- [ ] 无通知横幅、通知中心、短信或验证码
- [ ] 无具体地址、门牌、实时位置、常用路线或定位信息
- [ ] 视频声音中无姓名、电话、地址、账号、对话等隐私信息
- [ ] 文件元数据中无定位、设备所有者姓名等敏感信息

未经本人逐项确认，不要自动发布包含以上信息的素材。
